# 🏥 MediScan AI — Smart Health Assistant

MediScan AI is a modern, AI-powered web application designed to assist users in analyzing medical reports and understanding health conditions easily. It provides intelligent insights for blood reports, X-ray scans, and symptoms along with basic home remedy suggestions.

# 🚀 Features
### 🩸 Blood Report Analyzer

Upload reports (PDF, JPG, PNG, CSV, TXT)

Automatically detects abnormal parameters

Provides simplified medical insights

Includes demo report for testing

### X-Ray / CT / MRI Analyzer

Supports X-ray, CT scan, MRI, and ultrasound reports

Region-by-region AI-based analysis

Demo reports (Normal & Abnormal cases)

Suggests whether doctor consultation is needed

### 🩺 Symptom Checker

Select multiple symptoms

Get instant home remedy suggestions

Simple and user-friendly interface

# 🔐 Authentication System

User Signup & Login

Input validation with error handling

Personalized dashboard after login

# 📊 Dashboard

Centralized health analysis panel

Interactive UI with multiple cards

Displays results dynamically

## 🔒 Privacy & Security

Data processed locally in browser

No external data sharing

# 🖥️ Tech Stack
